编写函数 `evaluate_agent_on_benchmark(agent_outputs, ground_truth)`：
- 计算 Pass Rate：agent_output 与 ground_truth 完全匹配的比例
- 计算 Partial Score：部分匹配得 0.5，完全匹配得 1.0
- 返回 {"pass_rate": float, "partial_score": float, "total": int, "passed": int}

【输入输出说明】
输入：agent_outputs(list[str]), ground_truth(list[str])
输出：评分报告 dict

【初始代码】
```python
def evaluate_agent_on_benchmark(agent_outputs, ground_truth):
    """
    评估Agent在基准测试上的表现。
    """
    total = len(ground_truth)
    passed = 0
    partial = 0.0
    # TODO: 计算 pass_rate 和 partial_score
    for pred, gt in zip(agent_outputs, ground_truth):
        if pred == gt:
            passed += 1
            partial += 1.0
        elif gt in pred or pred in gt:
            partial += 0.5
    return {
        "pass_rate": round(passed / total, 3) if total else 0,
        "partial_score": round(partial / total, 3) if total else 0,
        "total": total,
        "passed": passed
    }
```

关卡 6-2：CRAG Benchmark RAG 评估
所属模块：模块六 · 评估、安全与前沿拓展
--------------------------------------------------------------------------------

【任务描述】
CRAG（Comprehensive RAG Benchmark）专门评估 RAG 系统。核心指标：
- 检索准确率（Retrieval Accuracy）：检索到的文档中有多少是相关的
- 生成准确率（Generation Accuracy）：最终答案的正确率

编写函数 `evaluate_rag_system(retrieved_docs, relevant_docs, answers, ground_truth)`：
- 计算 Recall@k（检索对相关文档的覆盖率）
- 计算 Answer Accuracy（答案完全匹配率）

【输入输出说明】
输入：retrieved_docs(list[list[str]]), relevant_docs(list[set[str]]), answers(list[str]), gt(list[str])
输出：{"recall": float, "accuracy": float}

【初始代码】
```python
def evaluate_rag_system(retrieved_docs, relevant_docs, answers, ground_truth):
    """
    评估RAG系统的检索和生成质量。
    """
    n = len(ground_truth)
    # Recall: 每个查询检索到的相关文档占比
    recalls = []
    for ret, rel in zip(retrieved_docs, relevant_docs):
        if rel:
            hits = len(set(ret) & rel)
            recalls.append(hits / len(rel))
        else:
            recalls.append(1.0)
    avg_recall = sum(recalls) / n if n else 0
    # Accuracy: 答案正确率
    correct = sum(1 for a, g in zip(answers, ground_truth) if a.strip() == g.strip())
    return {
        "recall": round(avg_recall, 3),
        "accuracy": round(correct / n, 3) if n else 0
    }
```

【评测代码】
```python

    gt = ["答案A", "答案C"]
```

- 计算：avg_reasoning_depth（平均推理步数）、tool_accuracy（正确调用比例）、
  task_completion_rate（完成任务比例）

【输入输出说明】

```python
def multi_dimension_eval(results):
    """
    多维度评估智能体能力。
    """
    n = len(results)
    if n == 0:
        return {"avg_reasoning_depth": 0, "tool_accuracy": 0, "completion_rate": 0}
    # TODO: 计算三个维度
    avg_depth = sum(r.get("reasoning_steps", 0) for r in results) / n
    tool_ok = sum(r.get("tool_calls_correct", 0) for r in results)
    tool_total = max(sum(r.get("tool_calls_total", 0) for r in results), 1)
    tool_acc = tool_ok / tool_total
    done = sum(1 for r in results if r.get("task_done", False))
    return {
        "avg_reasoning_depth": round(avg_depth, 1),
        "tool_accuracy": round(tool_acc, 3),
        "completion_rate": round(done / n, 3)
    }
```

```python
def check_tool_permission(tool_name, user_role, permission_policy):
    """
    检查用户是否有权限调用指定工具。
    """
    # TODO: 查找策略，判断角色是否在允许列表中
    allowed_roles = permission_policy.get(tool_name, [])
```

```python
def reduce_hallucination(query, knowledge_base, response):
    """
    检测并标记回答中的疑似幻觉内容。
    """
    kb_words = set(knowledge_base.lower().split())
    resp_words = response.lower().split()
    suspicious = []
    # TODO: 找出response中不在knowledge_base中的长词（>3字符）
    for word in set(resp_words):
        if len(word) > 3 and word not in kb_words and word not in query.lower():
            suspicious.append(word)
    return {
        "clean": len(suspicious) == 0,
        "suspicious": suspicious[:5],
        "verified_response": response if len(suspicious) == 0 else response + " [引用已核查]"
    }
```

编写函数 `mask_sensitive_info(text)`：

```python
import re

def mask_sensitive_info(text):
    """
    对手机号、身份证、邮箱进行脱敏处理。
    """
    # TODO: 正则替换
    # 手机号: 1[3-9]\d{9} → 前3位+****+后4位
    text = re.sub(r'(1[3-9]\d)\d{4}(\d{4})', r'\1****\2', text)
    # 身份证: \d{17}[\dXx] → 前1位+16星+末位
    text = re.sub(r'(\d)\d{16}(\d[\dXx])', r'\1****************\2', text)
    # 邮箱: 用户名部分只保留首字母
    text = re.sub(r'(\w)[\w.]*(@\w+\.\w+)', r'\1***\2', text)
    return text
```

关卡 6-7：算法偏见检测与缓解
所属模块：模块六 · 评估、安全与前沿拓展
--------------------------------------------------------------------------------

【任务描述】
算法偏见（Bias）指模型对特定群体有系统性歧视。简单检测方法：对比不同群体
的模型输出差异。缓解方法：对输出进行均衡化处理。

编写函数 `detect_bias(predictions, group_labels)`：
- 计算每个群体的平均得分
- 如果群体间最大差异 > 阈值（0.15），返回 bias_report
- 返回 {"has_bias": bool, "group_scores": dict, "recommendation": str}

【输入输出说明】
输入：predictions(list[dict 含 score 和 group]), threshold(float)
输出：偏见检测报告

【初始代码】
```python
def detect_bias(predictions, threshold=0.15):
    """
    检测模型输出中的群体偏差。
    predictions: [{"score": 0.8, "group": "A"}, ...]
    """
    # TODO: 分组统计平均分，检查组间差异
    groups = {}
    for p in predictions:
        g = p["group"]
        if g not in groups:
            groups[g] = []
        groups[g].append(p["score"])
    avg_scores = {g: sum(s)/len(s) for g, s in groups.items()}
    scores_list = list(avg_scores.values())
    max_diff = max(scores_list) - min(scores_list) if scores_list else 0
    return {
        "has_bias": max_diff > threshold,
        "group_scores": avg_scores,
        "recommendation": "需要调整模型以消除偏差" if max_diff > threshold else "偏差在可接受范围"
    }
```

【评测代码】
```python

    ]
    r = student_func(preds, threshold=0.15)
    assert r["has_bias"], "A组≈0.89, B组≈0.61, 差异>0.15应检测到偏见"
```

关卡 6-8：具身智能（Embodied AI）感知-操作循环
所属模块：模块六 · 评估、安全与前沿拓展
--------------------------------------------------------------------------------

【任务描述】
具身智能体（Embodied Agent）在物理环境中感知和操作。核心循环：感知环境
→ 解析状态 → 规划动作 → 执行动作 → 观察效果。

编写函数 `embodied_agent_loop(environment, policy, max_steps=5)`：
- environment：dict，包含 state 和 actions 映射
- policy：函数，根据 state 返回 action_name
- 模拟循环直到 state 变为 "goal" 或超步数
- 返回执行轨迹列表

【输入输出说明】
输入：environment(dict), policy(callable), max_steps(int)
输出：[{"step": int, "state":str, "action":str}, ...]

【初始代码】
```python
def embodied_agent_loop(environment, policy, max_steps=5):
    """
    具身智能体感知-规划-行动循环。
    """
    trajectory = []
    state = environment.get("initial_state", "start")
    for step in range(1, max_steps + 1):
        action = policy(state)
        trajectory.append({"step": step, "state": state, "action": action})
        # 执行动作，更新状态
        transitions = environment.get("transitions", {})
        state = transitions.get((state, action), state)
        if state == "goal":
            trajectory.append({"step": step + 1, "state": "goal", "action": "done"})
            break
    return trajectory
```

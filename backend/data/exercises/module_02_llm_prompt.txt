编写函数 `react_loop(task, tools, max_rounds=5)` 模拟 ReAct 循环：

- 任务示例：计算 "(3+5)*2"，tools={"calc": lambda x: eval(x)}
- 返回每一轮的日志列表

【输入输出说明】
输入：task(str), tools(dict), max_rounds(int)
输出：list[dict]，每轮 {"Thought":..., "Action":..., "Observation":...}

【初始代码】
```python
def react_loop(task, tools, max_rounds=5):
    """
    模拟 ReAct 交替推理与行动。
    """
    log = []
    # TODO: 实现 Thought → Action → Observation 循环
    for i in range(max_rounds):
        thought = f"第{i+1}轮推理：需要调用工具来完成任务"
        # 选择第一个可用工具
        tool_name = list(tools.keys())[0]
        action_result = tools[tool_name](task)
        observation = str(action_result)
        log.append({"Thought": thought, "Action": f"调用{tool_name}", "Observation": observation})
        task = observation  # 更新task为观察结果
    return log
```

所属模块：模块二 · 大模型与提示词工程

【任务描述】
模型量化（Quantization）通过降低模型权重的数值精度来减少显存占用和加速推理。
常见精度：FP32（32位）→ FP16（16位）→ INT8（8位）→ INT4（4位）。

编写函数 `simulate_quantization(weights, bits)`：
- weights：浮点数列表
- bits：目标位数（32, 16, 8, 4）
- 模拟量化：计算缩放因子 scale = max(|w|) / (2^(bits-1)-1)，将每个权重
  四舍五入到最近的量化级别，再反量化回浮点
- 返回 {"quantized": [...], "dequantized": [...], "error": 平均绝对误差}

【输入输出说明】
输入：weights(list[float]), bits(int)
输出：dict 包含量化值、反量化值、平均误差

【初始代码】
```python
def simulate_quantization(weights, bits=8):
    """
    模拟权重量化与反量化过程。
    """
    max_val = max(abs(w) for w in weights)
    levels = 2 ** (bits - 1) - 1  # 量化级别数
    scale = max_val / levels if levels > 0 else 1.0
    # TODO: 量化（round(w/scale)*scale），计算误差
    quantized = []
    dequantized = []
    for w in weights:
        q = round(w / scale) * scale
        quantized.append(q)
        dequantized.append(q)
    error = sum(abs(weights[i] - dequantized[i]) for i in range(len(weights))) / len(weights)
    return {"quantized": quantized, "dequantized": dequantized, "error": error}
```

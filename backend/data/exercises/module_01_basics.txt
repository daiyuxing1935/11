关卡 1-9：智能体的基本运行闭环 Observe→Think→Act
所属模块：模块一 · 智能体基础通识
--------------------------------------------------------------------------------

【任务描述】
AI 智能体的核心运行闭环：观察（Observe）→ 思考（Think）→ 行动（Act），
简称 OTA 循环。每一步都依赖上一步的输出作为输入。

编写函数 `ota_loop(environment, max_steps=5)`，模拟 OTA 循环：
- Observe：读取环境当前状态
- Think：根据状态生成计划（plan）
- Act：执行计划中的行动，更新环境
- 循环直到目标达成或达到 max_steps
- 返回执行日志列表，每步格式："Step N: Observe=X → Think=Y → Act=Z"

【输入输出说明】
输入：environment（dict，包含 "state" 和 "goal"），max_steps（int）
输出：list[str]，每步的日志记录
样例输入：{"state": "idle", "goal": "completed"}, max_steps=3
样例输出：["Step 1: Observe=idle → Think=start → Act=processing", "Step 2: Observe=processing → Think=finish → Act=completed"]

【初始代码】
```python
def ota_loop(environment, max_steps=5):
    """
    模拟 Observe → Think → Act 循环。
    environment: {"state": ..., "goal": ...}
    返回日志列表
    """
    log = []
    for step in range(1, max_steps + 1):
        # 1. Observe
        observed = environment.get("state", "unknown")
        # 2. Think: 根据观察和目标决定计划
        if observed == environment.get("goal"):
            break
        plan = "advance" if observed != environment.get("goal") else "stay"
        # 3. Act: 执行计划
        environment["state"] = environment.get("goal") if plan == "advance" else observed
        # TODO: 完善上述逻辑并生成日志字符串
        log.append(f"Step {step}: Observe={observed} → Think={plan} → Act={environment['state']}")
    return log
```

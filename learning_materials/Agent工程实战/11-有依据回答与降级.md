def format_context(evidence):
    return "\n\n".join(
        f"[来源:{item['id']}]\n{item['text']}"
        for item in evidence[:3]
    )

## 四、代码层先做空证据判断

```python
def grounded_node(state):
    evidence = state.get("evidence", [])
    if not evidence:
        return {
            "answer": "暂未找到可靠依据，已为你转接人工客服。",
            "citations": [],
            "needs_human": True,
        }

    response = (grounded_prompt | model).invoke({
        "question": state["question"],
        "context": format_context(evidence),
    })
    return {
        "answer": str(response.content),
        "citations": [item["id"] for item in evidence[:3]],
        "needs_human": False,
    }
```

## 七、常见错误

1. 把检索到但没用于回答的文档也列为引用。
2. 空检索时仍调用模型“尝试回答”。
3. 捕获所有异常后返回“系统错误”，没有业务降级。
4. 引用只显示文件名，无法定位具体文档块。
